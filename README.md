# Umurava Web

Next.js frontend for the Umurava HR AI backend.

## Features
- Dashboard overview
- AI ranking runner
- Candidate JSON upload
- Candidates with linked jobs
- Jobs list

## Setup
1. Install dependencies inside `web/`
2. Set `NEXT_PUBLIC_API_BASE_URL=https://umurava-jaq4.onrender.com`
3. Run `npm run dev`

## Notes
This app expects the Nest API to be running on port 3000.
